# smart-ralph

Self-correcting supervisor for the `ralph` bash orchestrator. Claude subscription only, no Docker, no API keys.

## Project decisions

Architectural decisions and rationale live in `docs/decisions/`. Roadmap is `docs/ROADMAP.md`. Full PRD is GitHub issue #1. Read the relevant document when a design choice comes up.

## Engineering standards

Apply KISS, YAGNI, DRY, SOLID, 12-factor, Clean Code. Use the `tdd` skill for all code work — red-green-refactor, vertical slices. If a test is hard to write, refactor toward testable shape first. No emojis in code or logs unless explicitly requested.
